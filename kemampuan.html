Kemampuan

Berikut adalah beberapa kemampuan yang saya miliki dalam bidang pengembangan web dan multimedia. Tingkat kemampuan ini menggambarkan sejauh mana saya memahami dan mampu mengaplikasikan setiap skill.

HTML (Menengah)

Mampu membuat struktur dasar halaman web yang semantik dan rapi. Terbiasa menggunakan elemen form, tabel, dan navigasi serta memahami struktur dokumen HTML5.

CSS (Menengah)

Menguasai desain tampilan website dengan CSS, menggunakan Flexbox dan Grid. Mampu membuat tampilan responsif dan menambahkan animasi serta efek hover untuk memperindah antarmuka.

PHP (Menengah)

Mengerti dasar pemrograman backend menggunakan PHP. Dapat membuat fitur CRUD, form input, koneksi database MySQL, serta memahami struktur logika sederhana.

Laravel (Menengah)

Menggunakan framework Laravel untuk membuat aplikasi web sederhana berbasis MVC. Memahami routing, controller, dan view, serta mampu membuat sistem login dan CRUD data.

JavaScript (Pemula)

Memahami dasar pemrograman JavaScript untuk menambah interaksi pada halaman web. Dapat memanipulasi DOM, membuat validasi form, dan efek sederhana seperti toggle menu.

Figma (Menengah)

Menggunakan Figma untuk membuat desain antarmuka (UI) dan prototype website. Mampu menyusun layout, memilih warna, serta membuat desain responsif dan konsisten antar halaman.
